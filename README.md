## 这个软件干什么的

翼课校长是一款专门为翼课学习中心考试任务打造的答案提取工具。

登录翼课账号后，翼课校长会从翼课接口读取学习中心考试列表，并根据考试详情、成绩信息和模型分数数据解析标准答案，让你可以在 App 内直接查看。

**主要功能：**
- 读取当前学习中心考试任务
- 读取历史学习中心考试任务
- 支持账号密码登录和实名登录
- 支持学校搜索、同名账号选择
- 获取试卷列表后自动在后台解析答案
- 解析完成后点击试卷查看标准答案
- 支持复制答案文本和分享答案图片
- 支持公告、更新提醒和远程管理
- 使用 miuix 风格界面组件

## 怎么用

### 第一步：先装上

1. 打开 [releases 页面](https://github.com/laststudio/eklose/releases)
2. 下载最新版本的 APK
3. 安装到你的手机上
4. 打开翼课校长 App

如果手机提示“禁止安装未知来源应用”，按照系统提示允许安装即可。

### 第二步：登录翼课账号

翼课校长目前支持两种登录方式：

| 方式 | 推荐程度 | 适合什么情况 |
|------|----------|--------------|
| 实名登录 | ⭐⭐⭐ 最推荐 | 只知道姓名、学校和密码 |
| 账号密码登录 | ⭐⭐ 稳定 | 已经知道翼课 username 和密码 |

简单说：
- 大多数情况优先用 **实名登录**
- 如果你手里有明确的账号 username，可以用 **账号密码登录**

### 第三步：按你的方式登录

#### 实名登录

1. 进入“主页”
2. 点击“登录状态”
3. 登录模式选择“实名登录”
4. 输入学校关键字
5. 点击搜索按钮
6. 在学校搜索结果里选择你的学校
7. 输入学生姓名和密码
8. 点击“实名登录”
9. 如果出现同名账号，选择对应账号继续登录

#### 账号密码登录

1. 进入“主页”
2. 点击“登录状态”
3. 登录模式选择“账号密码登录”
4. 输入账号 username 和密码
5. 点击“账号密码登录”

登录成功后会回到主页，登录状态会显示 uid。

### 第四步：读取考试任务

1. 进入底部“读取”页面
2. 在“读取考试”里选择任务来源
3. 点击右下角读取按钮
4. 等待考试列表加载
5. 列表加载后，翼课校长会在后台自动解析答案

当前支持的任务来源：

| 来源 | 说明 |
|------|------|
| 当前学习中心考试任务 | 读取学习中心当前考试任务 |
| 历史学习中心考试任务 | 读取学习中心历史考试任务 |

### 第五步：查看答案

读取成功后，试卷会显示在列表里。

刚读取出来的试卷可能会显示“等待解析”或“正在解析”，这时先等一下。等状态变成可点击后，点进试卷就能查看里面的标准答案。

答案页右上角支持：
- 复制答案文本
- 分享答案图片

如果没有读出来，可以先检查：
- 翼课账号是否已经登录成功
- 当前账号是否有学习中心考试任务
- 读取的是当前任务还是历史任务
- 网络是否正常
- 历史考试是否需要在设置里开启全分页读取
- App 是否是最新版本

## 云端读取详解

翼课校长不读取手机本地翼课 App 数据目录，而是使用翼课账号登录后，从翼课云端接口获取学习中心考试任务、考试详情、成绩信息和模型分数数据，再在本机解析标准答案。

**适合这些情况：**
- 不想折腾本地文件权限
- 手机没有安装翼课官方客户端
- 想直接查看当前或历史学习中心考试任务
- 想让 App 自动后台解析答案

**使用步骤：**
1. 打开 App，进入“主页”
2. 点击“登录状态”
3. 选择实名登录或账号密码登录
4. 登录成功后进入“读取”页面
5. 选择“当前学习中心考试任务”或“历史学习中心考试任务”
6. 点击右下角读取按钮
7. 等待考试列表加载和后台解析完成
8. 点击某个试卷查看答案

**重要风险提示：**
- 云端读取需要使用你的翼课账号登录翼课接口
- 读取前请确认没有正在进行考试、练习、录音或提交
- 如果翼课官方客户端登录状态发生变化，请重新登录
- 建议先用翼课校长读取并查看答案，再去翼课官方客户端完成任务
- 如果已经在翼课里开始考试或练习，不要中途频繁切换账号或重复读取

**权限说明：**
- 云端读取需要网络权限
- 云端读取不需要本地文件读取权限
- 云端读取不需要 Root
- 云端读取不需要 Shizuku

---

**关于数据安全：**
- 翼课校长通过翼课账号访问翼课接口
- 登录成功后会在本机保存 uid/token
- 如果在设置中开启“保存登录状态”，还会保存登录信息和密码
- 不需要保存密码时，可以关闭“保存登录状态”
- 点击主页“登录状态”的已登录项可以清除当前登录状态

## 注意事项

- 本软件仅供学习交流使用，请勿用于考试作弊等违规行为
- 使用前请确保你拥有对应翼课账号和学习内容的合法使用权限
- 翼课接口更新后软件可能会暂时无法使用，请耐心等待新版本发布
- 读取逻辑以学习中心考试任务为核心，不要把普通作业接口当成考试答案来源
